OS X Key Bindings (Default)

This page lists the default key bindings on Mac OS X. There are also pages that list the default key bindings on Windows/Linux. On any platform, you can also choose to use Emacs key bindings. As any of these key bindings schemes can be customized, these pages stand as references to the defaults. To view the current list of key bindings for your Komodo installation (including any custom key bindings), select Help|List Key Bindings. The different key binding schemes are either set according to the platform on which you installed Komodo, or they are selected in Edit|Preferences|Editor|Keybindings.

Note: The "Meta" key referred to in the following list may be better known to Mac OS X users as the "Command" key.
